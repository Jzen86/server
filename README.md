# Скрипт быстрой настройки SSH на чистом сервере

> ⚠️ **ВНИМАНИЕ ДЛЯ ВСЕХ, КТО НАШЕЛ ЭТОТ РЕПОЗИТОРИЙ ЧЕРЕЗ ПОИСК!**  
> Этот скрипт жестко зашивает **конкретный публичный SSH-ключ** автора и полностью отключает вход по паролю (`PasswordAuthentication no`).  
> Если вы запустите команду ниже без изменений, вы **ПОЛНОСТЬЮ ЗАБЛОКИРУЕТЕ** себе доступ к своему серверу!

---

## Что делает скрипт:
1. Обновляет пакеты системы (`apt update && apt upgrade`).
2. Создает директорию `.ssh` и файл `authorized_keys` с правильными правами доступа.
3. Прописывает публичный SSH-ключ для авторизации.
4. Добавляет в самый верх `/etc/ssh/sshd_config` правила безопасности (вход только по ключу, запрет паролей для root, защита сессии от обрывов).
5. Перезапускает службу SSH.

## Как правильно использовать под себя:
1. Сделайте **Fork** (копию) этого репозитория к себе в профиль.
2. Откройте файл `setup.sh` и замените значение переменной `PUBLIC_KEY="..."` на **свой собственный** публичный ключ.
3. Запустите на своем чистом сервере команду:
   ```bash
   wget -qO- [https://raw.githubusercontent.com/ВАШ_НИК/server/main/setup.sh](https://raw.githubusercontent.com/ВАШ_НИК/server/main/setup.sh) | sudo bash
